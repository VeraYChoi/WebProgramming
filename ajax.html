This API produces random facts about numbers which can have the category math, trivia, or be related to a date or year.

I found this at the website NumbersAPI.

In my API request, I had the option to specify whether I wanted a fact about a specific number, a range of numbers, or a random number. I also had the option to specify a fact related to math, trivia, a date, or a year. I have chosen to display a trivia and math fact about 17, the best number.

This API could be helpful if someone wanted to display a random number fact related to the current day on a website, or if someone wanted to make a website that lets anyone find a number fact related to their birthday.
