Are you bored? Here's a recreational activity!


Do you have a friend? Here's an activity for two participants!



This API gives activities that you can do. These activities can be completely random, be sorted by type, number of participants, price/price range, and accessibility/accessibility range.

I found this at the website https://www.boredapi.com/.

In my API request, I specified a recreational type activity for my first call and 2 participants for my second call. I could have requested a different type of activity like 'educational' or a different number of people.

This API could be helpful if someone were bored and wanted to find an activity to do that would also be productive and educational, or perhaps if a group of people of a specific number wanted to find an activity to do together.
